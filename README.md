# 运维平台

## 前端使用 vite+vue3+Element ui+axios+pinia+TS
* 封装axios  使用pinia 本地存储 用户token
## 后端使用 gin+jwt+gorm
* jwt 签发用户 token
* 对 cloudflare 域名的增删改查
* 对 包网信息的增删改查
* 修改腾讯云cos存储桶文件
* 检测域名ICP备案
* 接入亚洲云海cdnApi进行增删改查
* 接入阿里云 SDK
